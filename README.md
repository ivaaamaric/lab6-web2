# cookbook

Pokrenuti i spa-backend koji vraća vježbe, dakle:

...\spa-backend>node server.js
...\spa>npm run serve


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
